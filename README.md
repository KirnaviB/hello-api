# Hello World API

## Description
A beginner-friendly Express.js API created to understand the basics of
Node.js, Express, routing, and JSON responses.

This project covers the fundamentals of building and running a web server
and handling multiple API endpoints.

## Endpoints

- GET /hello  
  Returns a simple "Hello World!" message.

- GET /time  
  Returns the current server time in JSON format.

- GET /random  
  Returns a random number between 0 and 99.

- GET /greet/:name  
  Returns a personalized greeting using a route parameter.

## Technologies Used
- Node.js
- Express.js

## Learning Outcomes
- Setting up an Express server
- Understanding npm and package.json
- Creating API routes
- Using route parameters
- Returning JSON responses
- Testing APIs in the browser
